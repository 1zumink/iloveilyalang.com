# iloveilyalang.com — портфолио Ильи Ланга

Статический сайт (HTML + CSS + ванильный JS), без сборки. Открывается просто как `index.html`.

## Главное правило: никаких px

**Все размеры — только в `vw` / `vh`, в крайнем случае `%`.** Сайт масштабируется пропорционально на любую ширину экрана, поэтому фиксированные пиксели запрещены везде: CSS, инлайн-стили, JS-трансформации.

- Десктопный макет — 1440px: `1px макета = 100/1440 vw ≈ 0.0694vw`.
- Мобильный макет (медиазапрос `max-width: 700px`) — 375px: `1px макета = 100/375 vw ≈ 0.2667vw`.
- В JS сдвиги тоже считаются в `vw` (скролл переводится через `scrollY / innerWidth * 100`) или в `%` от размера элемента.
- Исключения: `line-height` без единиц, углы (`deg`), время (`s`/`ms`), безразмерные коэффициенты.

## Структура

- `index.html` — главная; `case-*.html` — страницы кейсов (общие шапка/футер/стили).
- `stylesheets/style.css` — только `@import` остальных: `reset`, `fonts`, `layout` (структура), `animation` (всё движение: темы, reveal, ховеры), `adaptive` (мобилка).
- `javascripts/scripts.js` — один файл на все страницы: бургер, смена темы по `data-theme`, reveal по скроллу (на мобилке элементы въезжают сбоку — кейфреймы в adaptive.css), параллакс (десктоп — мышь+скролл, мобилка — скролл-параллакс фото героя), магнитные кнопки, наклон карточек, счётчики цен, кастомный курсор-точка и трейл-«комета» за курсором на 2D-канвасе (цвет обоих следует за темой: белый → акцент, розовый → чёрный, тёмный → белый; только для мыши).
- `javascripts/infiniteMenu.js` — только на главной: 3D-сфера кейсов (порт React Bits `<InfiniteMenu />` на ванильный JS + WebGL2). Требует `javascripts/vendor/gl-matrix-min.js`. Если WebGL2/gl-matrix недоступны или включён `prefers-reduced-motion`, секция кейсов остаётся с обычной сеткой `.cases__grid` (класс `.cases--menu` на секции переключает варианты).

## Конвенции

- Тексты, комментарии в коде — на русском.
- Смена фона по скроллу — зонами: тема берётся у последней секции с `data-theme="pink|dark"`, чей верх прошёл центр экрана, и действует до следующей такой секции (без белых промежутков). Сейчас: герой, «Обо мне» и «Достижения» белые; «Как я работаю» и кейсы розовые; резюме, цены и контакты тёмные. Классы `theme-pink`/`theme-dark` на `<body>`, стили в animation.css.
- Тяжёлые эффекты выключаются при `prefers-reduced-motion` и на тач-устройствах (`pointer: fine` проверяется в JS). Все CSS-ховеры собраны в animation.css в блоке `@media (hover: hover) and (pointer: fine)` — на таче их нет.
- Мобильные отступы секций — 8vw с каждой стороны, блоки по центру.
- Заглушки помечаются комментарием `ЗАГЛУШКА: ...`.
